# Занятие 1. Базовые операции и синтаксис Python

## Краткая сводка (без кода)

**1. Установка Python 3.** С официального сайта python.org берут установщик (для 64-битной Windows — x86-64 executable installer). При установке важно отметить добавление Python в PATH. После установки доступна среда IDLE.

**2. IDLE.** Режим по умолчанию — интерактивная консоль. File → New File — новый скрипт, File → Open — открыть существующий .py. В окне редактора отображаются имя файла, путь, версия Python; внизу — номер строки (Ln) и столбца (Col). Несохранённые изменения помечаются «*». Файлы сохраняют с расширением .py.

**3. Работа в консоли и в файле.** Программы — это файлы .py с кодом. IDLE даёт подсветку синтаксиса, автоотступ; профессиональные IDE (например, VS Code, PyCharm) расширяют возможности.

**4–7.** Ниже — код: чтение CSV вложенным списком, расчёт среднего по колонке Age циклом, расчёт среднего отдельно для мужчин и женщин (if-else), построение таблицы «Диабет × Ожирение» (четыре группы пациентов по номерам строк).

## Суть заданий для защиты

- **CSV через `csv`:** данные читаются как «список списков»; первая строка — заголовки, дальше строки таблицы.
- **Среднее по колонке:** перебор строк (или индексов), суммирование значений нужной колонки, деление на количество (учесть, что заголовок не число).
- **Условие по полу:** в цикле по строкам проверять колонку пола (Male/Female), накапливать суммы и счётчики отдельно для мужчин и женщин.
- **Таблица диабет–ожирение:** у каждого пациента есть метки «диабет» (Positive/Negative) и «ожирение» (Positive/Negative). Два цикла или одно прохождение с if-else дают четыре списка индексов (диабет+/ожирение+, диабет+/ожирение-, и т.д.); их выводят в виде таблицы 2×2.
