# Занятие 4. Работа с табличными данными в библиотеке Pandas

## Краткая сводка (без кода)

**Структуры Pandas.** Series — одномерная маркированная структура (доступ по индексу и по метке). DataFrame — двумерная таблица с именованными строками и столбцами; создание и работа аналогичны таблицам в R.

**Импорт/экспорт.** Для загрузки CSV: `pd.read_csv('файл.csv')`. Поддерживаются и другие форматы: Excel, JSON, HTML, Parquet, SQL и т.д. (read_*/to_*).

**Исследование DataFrame.** `df.dtypes` — типы столбцов; `df.head(n)`, `df.tail(n)` — первые/последние n строк; `df.index` — метки строк; `df.to_numpy()` — перевод в NumPy; `df.describe()` — описательная статистика; `df.T` — транспонирование.

**Выборка.** `df['Age']` — столбец; `df[0:3]` — срезы строк; `df.loc[]` — по меткам; `df.iloc[]` — по целочисленным индексам; `df[df['A'] > 0]` — фильтр по условию.

**Сортировка.** `df.sort_values(by=['col1','col2'])`, `df.sort_index()`.

**Пропуски.** `pd.isna(df)` — поиск пропусков; `df.dropna(how='any')` — удаление строк с пропусками; `df.fillna(value)` — заполнение.

**Визуализация.** Через matplotlib: `plt.hist()`, `plt.boxplot()`, `plt.subplot()` для нескольких графиков на одном полотне. Scatter matrix: `pd.plotting.scatter_matrix()` или попарные графики с раскраской по категории (например, по классу).

## Суть для защиты

- Чем Series отличается от DataFrame; зачем нужны loc и iloc.
- Как отфильтровать строки по условию и отсортировать по нескольким столбцам.
- Как обрабатывать пропуски и зачем удалять или заполнять их перед обучением.
- Интерпретация гистограмм и boxplot по подвыборкам (например, по признаку Yes/No): сравнение распределений Age в двух группах.
