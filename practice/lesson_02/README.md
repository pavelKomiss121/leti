# Занятие 2. Загрузка и настройка модулей NumPy, Pandas, scikit-learn, TensorFlow, Keras

## Краткая сводка (без кода)

**Обзор библиотек.** NumPy — многомерные массивы и быстрые математические операции. Pandas — табличные данные (DataFrame), построен поверх NumPy. Scikit-learn — классификация, регрессия, кластеризация (использует NumPy/SciPy). TensorFlow и Keras — нейронные сети (TensorFlow от Google, Keras — высокоуровневый API поверх TensorFlow/Theano).

**Установка из командной строки.** `pip install <пакет>`, удаление — `pip uninstall`, обновление — `pip install --upgrade <пакет>`. На Windows может потребоваться запуск cmd от имени администратора.

**Импорт в скрипте.** Импортируют весь модуль (`import numpy as np`) или отдельные объекты (`from sklearn.linear_model import LogisticRegression`).

**Версия пакета из кода.** У большинства библиотек есть атрибут `__version__` (например, `np.__version__`, `pd.__version__`). Альтернатива — вывод `pip freeze` в консоли или вызов `pip show <пакет>`.

**Чтение CSV через Pandas.** `pd.read_csv('файл.csv')` возвращает DataFrame — таблицу с именованными колонками; доступ к колонке: `df['Age']`, среднее: `df['Age'].mean()`.

**Замер времени.** Модуль `time`: `start = time.time()`, выполнить код, `end = time.time()`, разница в секундах. Сравнение времени расчёта среднего вручную (цикл по списку) и через Pandas/NumPy показывает выигрыш векторных операций.

## Суть для защиты

- Pip служит для установки пакетов Python; `pip freeze` выводит версии установленных пакетов.
- Pandas упрощает загрузку CSV и расчёты по колонкам (mean, группировка по полу) по сравнению с ручным циклом по списку списков.
- Замер времени выполнения одного и того же расчёта разными способами иллюстрирует эффективность векторных библиотек (NumPy/Pandas против чистого Python).
