# Курсовая работа: Rain in Australia

Статистический анализ и моделирование данных на примере базы данных «Rain in Australia».

## Структура

- **data/** — поместите сюда файл датасета `weatherAUS.csv` (скачать с [Kaggle](https://www.kaggle.com/datasets/jsphyg/weather-dataset-rattle-package)).
- **figures/** — сохраняемые графики (гистограммы, scatter, boxplot, heatmap, ROC и т.д.).
- **текст/** — текстовый отчёт (ОТЧЕТ_КУРСОВАЯ.md), а также **ХОД_РАБОТЫ.md** — журнал действий, проблем и решений для быстрого написания отчёта и контроля прогресса.

**Запуск** (из корня репозитория `leti`):
```bash
python -m course_work.main
```

**Структура кода (один файл — одна задача):**
- `main.py` — только запуск пайплайна (вызов модулей, вывод итогов).
- `data.py` — загрузка и первичная обработка данных (проверка, очистка).
- `viz.py` — визуализация и графики (Задание 2).
- `correlation.py` — корреляционный анализ (Задание 3).
- `regression.py` — регрессионный анализ (Задание 4).
- `anova.py` — дисперсионный анализ ANOVA (Задание 6).
- `classifier.py` — классификаторы (Задание 7: LogisticRegression, DecisionTree, MLP, метрики, ROC).
- `utils.py` — общие вспомогательные функции (например, filter_iqr).

Анализ (корреляции, регрессия, классификация) по мере добавления заданий — в отдельные модули.
