## 1. Баланс классов (task2_class_balance.png)

**Что показано:** Доля дней, когда завтра будет дождь (Yes) и не будет (No).

**Вывод:** Классы несбалансированы: ~78% — «нет дождя», ~22% — «дождь». Модель может «поддаться» и чаще предсказывать «нет дождя», поэтому используются SMOTE и `class_weight='balanced'`.

## 2. Распределения выбранных признаков по классам (task2_distributions_by_class.png)

Три гистограммы: Humidity3pm, Rainfall, Humidity9am. Для каждого признака — распределение по классам RainTomorrow (No — синий, Yes — оранжевый).

### Humidity3pm
- **No:** пик в районе 50–60%, ниже влажность.
- **Yes:** пик около 80–90%, выше влажность.
- **Вывод:** Высокая влажность в 15:00 связана с дождём на следующий день. Признак хорошо разделяет классы.

### Rainfall
- **No:** почти все значения 0 мм.
- **Yes:** много нулей, но заметная доля ненулевых осадков.
- **Вывод:** Осадки сегодня чаще связаны с дождём завтра. Распределение сильно скошено вправо (много нулей).

### Humidity9am
- **No:** пик около 60–70%.
- **Yes:** пик около 80–90%.
- **Вывод:** Высокая утренняя влажность тоже связана с дождём завтра, но разделение слабее, чем у Humidity3pm.

## 3. Выше/ниже медианы (task2_above_below_median_*.png)

Для каждого признака: гистограмма (разделение по медиане) и boxplot по классам RainTomorrow.

### Humidity3pm
- **Гистограмма:** медиана ≈ 52%. Синие — ниже медианы, оранжевые — выше.
- **Boxplot:** медиана для «No» ≈ 48%, для «Yes» ≈ 69%. Ящики почти не пересекаются.
- **Вывод:** Humidity3pm хорошо различает дни с дождём и без.

### Rainfall
- **Гистограмма:** медиана = 0. Почти все точки — «ниже медианы» (синие), выше — редкие ненулевые осадки.
- **Boxplot:** для «No» ящик почти на нуле; для «Yes» — выше и шире.
- **Вывод:** Rainfall сильно скошен; при ненулевых осадках чаще ожидается дождь завтра.

### Humidity9am
- **Гистограмма:** медиана ≈ 70%.
- **Boxplot:** медиана для «No» ≈ 67%, для «Yes» ≈ 80%.
- **Вывод:** Утренняя влажность тоже помогает предсказывать дождь, но слабее, чем Humidity3pm.

## 4. Тепловые карты корреляций (task3_heatmap_pearson.png, task3_heatmap_spearman.png)

**Что показано:** Корреляция между парами признаков. Красный — положительная связь, синий — отрицательная. Чем ярче цвет, тем сильнее связь.

### Связь с RainTomorrow
- **Humidity3pm:** |r| ≈ 0.44 — самая сильная связь.
- **Rainfall:** |r| ≈ 0.29–0.33.
- **Humidity9am:** |r| ≈ 0.26–0.27.

**Вывод:** Эти три признака выбраны как наиболее связанные с целевой переменной.

### Мультиколлинеарность (яркие квадраты между признаками)
- MaxTemp и Temp3pm (0.97), Pressure9am и Pressure3pm (0.96) — почти дублируют друг друга.
- **Вывод:** В регрессии не стоит использовать все такие пары одновременно.

---

## 5. Регрессионный анализ (Задание 4)

**Humidity3pm и Rainfall:** R² ≈ 0.09 — слабая линейная связь. Связь есть, но не строго линейная.

**Humidity3pm и Humidity9am:** R² ≈ 0.44 — заметная линейная связь. Утренняя и дневная влажность согласованы.

**Rainfall и Humidity9am:** R² ≈ 0.07 — слабая связь.

---

## 6. ANOVA (Задание 6)

Для всех трёх признаков p-value = 0.0000: средние значения значимо различаются между классами «дождь» и «нет дождя».

- **Humidity3pm:** eta² = 0.195 — самый сильный эффект.
- **Rainfall:** eta² = 0.082.
- **Humidity9am:** eta² = 0.066.

**Вывод:** Все три признака статистически значимо связаны с RainTomorrow; Humidity3pm даёт наибольший эффект.
